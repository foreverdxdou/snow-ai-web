## PUT 更新大模型配置

PUT /api/v1/llm/config

> Body 请求参数

```json
{
  "id": 0,
  "modelName": "string",
  "apiUrl": "string",
  "apiKey": "string",
  "enabled": true,
  "createTime": "2019-08-24T14:15:22Z",
  "updateTime": "2019-08-24T14:15:22Z"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|[LlmConfig](#schemallmconfig)| 否 |none|

> 返回示例

> 200 Response
