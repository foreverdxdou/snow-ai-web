## GET 获取权限详情

GET /api/v1/system/permission/{id}

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|id|path|integer(int64)| 是 |权限ID|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "message": "string",
  "data": {
    "id": 0,
    "parentId": 0,
    "name": "string",
    "type": 0,
    "permissionCode": "string",
    "path": "string",
    "component": "string",
    "icon": "string",
    "sort": 0,
    "status": 0,
    "hasChildren": 0,
    "createTime": "2019-08-24T14:15:22Z",
    "updateTime": "2019-08-24T14:15:22Z"
  }
}
```
